# ACC102 Track 2: S&P 500 Stock Data Analysis
GitHub Data Analysis Project

## 1. Problem & User
This project analyzes simulated S&P 500 stock data to identify price trends, return patterns, and key statistical insights for investors and finance students.

## 2. Data
- Source: Simulated S&P 500 daily stock data
- Access Date: 2026-04-19
- Key Fields: Date, Open, High, Low, Close, Volume, Daily_Return

## 3. Methods
- Data generation and cleaning with pandas
- Daily return calculation
- Descriptive statistical analysis
- Time-series visualization with matplotlib

## 4. Key Findings
- Clear upward trend in S&P 500 price from 2023 to 2026
- Stable positive average daily return
- Identified highest and lowest closing prices
- Volatility remains within reasonable range

## 5. How to Run
1. Install dependencies: pip install -r requirements.txt
2. Open and run sp500_analysis.ipynb
3. View outputs, statistics, and plots
